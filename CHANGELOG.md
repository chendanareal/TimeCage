
# Changelog

所有项目重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，项目版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.1.0] - 2026-06-09

### 新增
- 自定义老板键锁定解锁游戏
- 支持最小化到托盘区
- 支持开机自启设置（默认为不开启）
- 可设置是否同时只启动一个程序

## [2.0.0] - 2026-05-30

### 新增
- 基于 WebView2 + HTML/CSS/JS 的现代化 UI 界面
- 支持 4 个快捷按钮配置
- 配置文件持久化（INI 格式）
- 冻结时窗口隐藏选项（普通/增强模式）
- DPI 自适应窗口缩放
- 深色科技感界面设计

### 改进
- 使用 nlohmann/json 库处理前后端通信
- 优化的进程管理逻辑

---

## [1.0.0] - (历史版本)

### 初始版本
- 基础的进程冻结/解冻功能
- 简单的 MFC 界面
